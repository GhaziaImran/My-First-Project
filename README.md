🎯 Number Guess Game

A simple console-based Number Guess Game built with Java, where players try to guess a randomly generated number between **1 and 100**.
The game provides helpful hints after each guess and tracks the total number of attempts until the correct number is guessed.

Project Overview:

This project demonstrates the fundamentals of Java programming through an interactive command-line game. 
It utilizes loops, conditional statements, user input handling, and random number generation to create a fun and beginner-friendly experience.

✨ Features

🎲 Random number generation between **1 and 100**
💬 Interactive command-line gameplay
📈 "Too High" and "Too Low" hints after every incorrect guess
🔢 Tracks the total number of attempts
🎉 Displays a congratulatory message upon winning
🧹 Clean and simple console interface

Technologies Used:
- Java
- Random Class
- Scanner Class

▶️ How to Run
1. Clone this repository.
2. Open the project in your preferred Java IDE (VS Code, IntelliJ IDEA, Eclipse, etc.).
3. Compile and run the `NumberGuessGame.java` file.
4. Enter your guesses and continue until you find the correct number.

🎮 Game Rules
- The computer randomly selects a number between **1 and 100**.
- Enter your guess when prompted.
- If your guess is:
  - **Greater** than the correct number → You'll receive a **"Too High!"** hint.
  - **Less** than the correct number → You'll receive a **"Too Low!"** hint.
- Continue guessing until you find the correct number.
- The game displays the correct number along with your total attempts after you win.
